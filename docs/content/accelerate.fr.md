Dans ce nouveau monde de l'IA, certaines entreprises cherchent à accélérer les développements ou l'ensemble du cycle de vie logiciel.

Cela peut passer par différents *frameworks* (liste non exhaustive) que cette documentation ne détaillera pas :
- **SDLC** : Software Development Life Cycle
- [**BMAD**](https://docs.bmad-method.org) : Build More Architect Dreams
- **SDD** : Spec Driven Development

*La plupart des frameworks mis en avant avec l'IA ne sont pas une révolution : ils existaient bien avant et étaient déjà utilisés, mais de façon implicite.*
