---
title: Command / Prompt
---

Commands (or prompts) were precursors of [skills](/components/skill),
the whole purpose was to be able to invoke custom instructions on-demand based on a slash command within an agent (e.g. `/name`).

However since skills, vendors tend to merge or deprecated commands (or prompts)
because skills achieve both this ability to invoke slash command but not just for the users but also the agent.

> [!tip]
> Create skills instead of commands or prompts.
