---
title: Command / Prompt
---

Commands (or prompts) were precursors of [skills](/components/skill): their whole purpose was to invoke custom instructions
on demand through a slash command within an agent (e.g. `/name`).

However, since skills emerged, vendors tend to merge or deprecate commands (or prompts), because skills provide that same
ability to invoke a slash command, not just for users but also for the agent.

> [!tip]
> Create skills instead of commands or prompts.
