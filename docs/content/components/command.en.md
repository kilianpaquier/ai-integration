---
title: Command / Prompt
---

Commands (or prompts) invoke custom instructions on demand through a slash command within an agent (e.g. `/name`).

However, with the standardization and adoption of skills, vendors tend to merge or deprecate commands (or prompts), because
skills provide that same ability to invoke a slash command, not just for users but also for agents.

> [!tip]
> Create skills instead of commands or prompts.
