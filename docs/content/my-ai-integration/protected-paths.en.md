---
title: protected-paths
---

Blocks tool calls from reading or writing credential and config directories (SSH keys, cloud credentials, kube/docker configs) via a PreToolUse hook.

- **Category:** Hooks
- **Source:** [`plugins/protected-paths`](https://gitlab.com/kilianpaquier/ai-marketplace/-/tree/main/plugins/protected-paths)

## Installation

```sh
claude plugin install protected-paths@bunch-of
```

```sh
copilot plugin install protected-paths@bunch-of
```

```sh
apm install protected-paths@bunch-of
```
