Official GitHub MCP server for repository management. Create issues, manage pull requests, review code, search repositories, and interact with GitHub's full API.

- **Category:** MCP
- **Source:** [`plugins/mcp/github`](https://gitlab.com/kilianpaquier/ai-marketplace/-/tree/main/plugins/mcp/github)
- **Upstream:** <https://github.com/github/github-mcp-server>

## Installation

```sh
claude plugin install github@bunch-of
```

```sh
copilot plugin install github@bunch-of
```

```sh
apm install github@bunch-of
```
