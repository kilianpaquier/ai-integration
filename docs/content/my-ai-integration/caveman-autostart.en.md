---
title: caveman-autostart
---

Automatically enables caveman's ultra-compressed communication mode at session start via a SessionStart hook that injects the vendored ruleset directly.

- **Category:** Hooks
- **Source:** [`plugins/caveman-autostart`](https://gitlab.com/kilianpaquier/ai-marketplace/-/tree/main/plugins/caveman-autostart)

## Installation

```sh
claude plugin install caveman-autostart@bunch-of
```

```sh
copilot plugin install caveman-autostart@bunch-of
```

```sh
apm install caveman-autostart@bunch-of
```
