# Context7 MCP

## Installation

```sh
claude plugin install context7@bunch-of
```

## MCP Servers

This plugin includes the Upstash Context7 MCP server, which resolves library IDs
and fetches version-specific documentation directly from source repositories.

Context7 source code lives on their [**GitHub**](https://github.com/upstash/context7).

## Compatibility

This plugin is expected to work with Claude and Copilot.
