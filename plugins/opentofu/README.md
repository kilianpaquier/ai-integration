# OpenTofu MCP

## Installation

```sh
claude plugin install opentofu@ki-marketplace
```

## MCP Servers

As the plugin title, this plugin includes the OpenTofu MCP server with its SSE mode.
It is pretty much only a wrapper to share the MCP configuration.

OpenTofu MCP server source code lives on their [**GitHub**](https://github.com/opentofu/opentofu-mcp-server).

## Compatibility

This plugin was tested with Claude and Copilot.
