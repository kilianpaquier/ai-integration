# TypeScript LSP

## Installation

Before being able to use this plugin, you must install [**typescript-language-server**](https://github.com/typescript-language-server/typescript-language-server) on your local system
since the LSP server runs locally.

```sh
claude plugin install typescript-lsp@bunch-of
```

```sh
copilot plugin install typescript-lsp@bunch-of
```

## LSP Servers

As the plugin title, this plugin includes the typescript-language-server LSP server with the local bin version.
It is pretty much only a wrapper to share the LSP configuration.

## Compatibility

This plugin is expected to work with Claude and Copilot.
