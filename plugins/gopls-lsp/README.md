# Gopls LSP

## Installation

Before being able to use this plugin, you must install [**gopls**](https://go.dev/gopls) on your local system
since the LSP server runs locally.

```sh
claude plugin install gopls-lsp@bunch-of
```

## LSP Servers

As the plugin title, this plugin includes the gopls LSP server with the local bin version.
It is pretty much only a wrapper to share the LSP configuration.

## Compatibility

This plugin is expected to work with Claude and Copilot.
