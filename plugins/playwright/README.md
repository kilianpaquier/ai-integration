# Playwright MCP

## Installation

```sh
claude plugin install playwright@bunch-of
```

## MCP Servers

This plugin includes the Microsoft Playwright MCP server, which enables browser
automation: navigating pages, clicking elements, filling forms, and taking screenshots.

Playwright MCP source code lives on their [**GitHub**](https://github.com/microsoft/playwright-mcp).

## Compatibility

This plugin is expected to work with Claude and Copilot.
