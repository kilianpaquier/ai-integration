# Codebase Memory MCP

## Installation

```sh
claude plugin install codebase-memory-mcp@bunch-of
```

```sh
copilot plugin install codebase-memory-mcp@bunch-of
```

## MCP Servers

As the plugin title, this plugin includes the Codebase Memory MCP server with the local bin version.
It is pretty much only a wrapper to share the MCP configuration.

Codebase Memory MCP source code (an MCP server source code) live on their [**GitHub**](https://github.com/DeusData/codebase-memory-mcp).

## Compatibility

This plugin is expected to work with Claude and Copilot.
