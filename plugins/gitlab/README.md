# GitLab MCP

## Installation

```sh
claude plugin install gitlab@bunch-of
```

## MCP Servers

This plugin includes the native GitLab MCP server, which enables repository
management, merge request workflows, issue tracking, and CI/CD pipeline management.

The MCP endpoint is part of the [**GitLab**](https://gitlab.com/gitlab-org/gitlab) platform.

## Compatibility

This plugin is expected to work with Claude and Copilot.
